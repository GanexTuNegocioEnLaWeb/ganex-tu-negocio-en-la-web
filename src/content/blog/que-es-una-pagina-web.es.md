> En esta guía te voy a explicar **qué es una página web**, para qué sirve, los tipos que existen y cómo puede ayudarte a **hacer crecer tu negocio en Internet** de forma estratégica.

## ¿Qué es una página web?

Imagina una _página web_ como una _página de un periódico tradicional_. No hablamos del periódico completo, sino de **una sola página**: un espacio individual que contiene información organizada.

Esa página tiene elementos muy similares a los de un periódico físico:

- Encabezados
- Títulos principales y secundarios
- Párrafos
- Imágenes
- Listas
- Enlaces

La diferencia es que una **página web vive en Internet** y se puede acceder a ella desde cualquier parte del mundo usando un navegador como Chrome, Firefox o Safari.

Cada página web es un **documento digital único**, pensado para informar, persuadir o generar una acción concreta (contactar, comprar, registrarse, etc.).

## ¿Cómo entiende el navegador una página web?

Los navegadores saben qué es cada elemento gracias al **Lenguaje de Marcado de Hipertexto**, más conocido como [HTML](https://developer.mozilla.org/es/docs/Web/HTML) (_HyperText Markup Language_).

HTML utiliza **etiquetas** para indicar qué tipo de contenido es cada parte del texto.

### Ejemplo básico de HTML

```html
<h1>Soy un encabezado principal</h1>
<h2>Soy un encabezado secundario</h2>
<p>Soy un párrafo de texto</p>
```

Estas etiquetas le dicen al navegador y a Google **qué es importante y qué no** dentro de una página web.

También es posible escribir texto sin etiquetas:

```html
Soy un texto sin etiqueta
```

### ¿Cuál es el problema de no usar etiquetas?

El mayor problema es el **SEO**.

**SEO (Search Engine Optimization)** es la optimización para motores de búsqueda como Google. Sin una estructura clara en HTML:

- Google no entiende el contenido
- Tu página no se posiciona bien
- Pierdes visibilidad y clientes potenciales

> 💡 **Dato importante:** Una página bien estructurada en HTML tiene muchas más posibilidades de aparecer en los primeros resultados de búsqueda.

## ¿Se puede crear una página web fácilmente?

> **¿Sabías que puedes crear una página web con un simple archivo de texto?**

Solo tienes que:

1. Crear un archivo `.txt` (más conocido como bloc de notas)
2. Escribir cualquier texto (por ejemplo: _Hola mundo_)
3. Guardarlo como `cualquier_nombre.html`

🎉 ¡Felicidades! Acabas de crear tu **primera página web**.

Claro que una página web profesional requiere mucho más: diseño, SEO, velocidad, experiencia de usuario y estrategia digital.

## Página web vs sitio web: diferencia clave

Muchas personas confunden estos conceptos:

| Término        | Definición                                |
| -------------- | ----------------------------------------- |
| **Página web** | Un solo documento HTML dentro de Internet |
| **Sitio web**  | Conjunto de varias páginas web conectadas |

Por ejemplo:

- La **página de inicio**
- La **página de blog**
- La **página de contacto**

Son _páginas web individuales_, pero juntas forman un **sitio web completo**.
